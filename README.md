# Jonathan Bardolf's Portfolio

A modern, responsive personal portfolio website showcasing projects, experiences, and professional information.

## Overview

This is a sleek, minimalist portfolio built with vanilla HTML and CSS. Features a smooth navigation menu, responsive design, and a clean aesthetic with a gradient background and frosted glass effects.

## Features

- **Responsive Design**: Adapts seamlessly to desktop, tablet, and mobile devices
- **Modern Styling**: Clean typography, smooth animations, and a professional color scheme
- **Navigation Menu**: Liquid glass-style dropdown navigation with smooth transitions
- **Smooth Animations**: Card fade-up animations and hover effects for enhanced interactivity
- **Accessibility**: Semantic HTML and proper ARIA attributes

## Pages

- **index.html** - Home page with introduction and quick links
- **about.html** - About me page with personal background
- **experiences.html** - Professional experiences and work history
- **projects.html** - Portfolio of completed projects
- **interests.html** - Hobbies, interests, and passions
- **contact.html** - Contact information and messaging
- **thank-you.html** - Thank you page for form submissions

## Getting Started

Simply open `index.html` in a web browser to view the portfolio locally, or deploy to a web server.

## Technology Stack

- HTML5
- CSS3 (with modern features like CSS Grid, Flexbox, and backdrop filters)
- JavaScript (for interactive menu)

## Design Highlights

- **Color Scheme**: Professional blues and grays with accent colors
- **Typography**: Inter font family with responsive sizing
- **Effects**: Backdrop blur, box shadows, and smooth transitions
- **Layout**: CSS Grid and Flexbox for responsive layouts
